引言

用 swagger2 来生成接口文档，让我们脱离了传统的手写文档的模式，它可以轻松的整合到 Spring Boot 中，用很少的代码就能生成一份漂亮的文档，同时让维护文档和修改代码整合为一体，可以让我们在修改代码逻辑的同时同步维护接口文档。另外 Swagger2 也提供了强大的页面测试功能来调试每个 RESTful API。具体效果如下图所示：
