首先看一下生成效果

和 Swagger 对比

优点

  • 和 Swagger 相比，JApiDocs 对现有代码无任何侵入，只需按 javadoc 格式添加注释，就能生成文档。

缺点

  • JApiDocs 适合生成静态文档，目前没有 Swagger 的调试功能。
  • 需要解析的对象必须在项目的源码中，不支持 Jar 包内的注释解析。
