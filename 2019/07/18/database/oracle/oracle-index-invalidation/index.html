4、 使用 LIKE ‘%T’ 进行模糊查询
5、 WHERE 子句中使用不等于操作

不等于操作包括：<>, !=, NOT colum >= ?, NOT colum <= ?

对于这个限制条件可以通过 OR 替代，例如：colum <> 0 ===> colum>0 OR colum<0

6、 等于和范围索引不会被合并使用

job 和 deptno 都是非唯一索引，这种条件下 oracle 不会合并索引，它只会使用第一个索引。
