Cron 表达式,由七个字段组成：

  • Seconds
  • Minutes
  • Hours
  • Day-of-Month
  • Month
  • Day-of-Week
  • Year (Optional field)

首先所有字段都有自己特定的取值,例如:

  • Seconds 和 Minutes 取值为 0 到 59,
  • Hours 取值为 0 到 23,
  • Day-of-Month 取值为 0-31,
  • Month 取值为 0-11,或者 JAN,FEB, MAR, APR, MAY, JUN, JUL, AUG, SEP, OCT, NOV, DEC.
  • Days-of-Week 取值为 1-7 或者 SUN, MON, TUE, WED, THU, FRI, SAT.
